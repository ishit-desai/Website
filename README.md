1. **Open Terminal/Command Prompt**
2. **Navigate to project directory**: `cd medical-website`
3. **Start a local server**:

   **Python (if installed):**
   ```bash

   npx serve .
   ```

   **PHP (if installed):**
   ```bash
   php -S localhost:8000
   ```
